# 🎮 Reinforcement Learning Foundations

> **Core concepts of learning from interaction**

---

## 📐 Key Concepts

```
Agent-Environment Loop:
Agent observes state s → takes action a → receives reward r → transitions to s'

Key Elements:
• State (s): Current situation
• Action (a): What agent can do
• Reward (r): Feedback signal
• Policy (π): Action selection strategy
• Value (V): Expected future reward
```
