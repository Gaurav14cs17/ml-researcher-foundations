## 💡 Key Formulas

<table>
<tr>
<td>

### 📐 Linear Algebra
```
Matrix:    (AB)ᵀ = BᵀAᵀ
Eigen:     A = QΛQᵀ
SVD:       A = UΣVᵀ
```

</td>
<td>

### 📈 Calculus
```
Gradient:  ∇f = [∂f/∂xᵢ]ᵀ
Chain:     ∂z/∂x = ∂z/∂y · ∂y/∂x
Jacobian:  J = [∂fᵢ/∂xⱼ]
```

</td>
<td>

### 🎯 Optimization
```
GD:   θ ← θ - η∇L(θ)
Adam: θ ← θ - η·m̂/(√v̂+ε)
```

</td>
</tr>
</table>
