## 🎯 What is Abstraction?

```
Abstraction = Ignoring irrelevant details to focus on essential structure

Example:
• Specific: "Add 2+3, multiply by 4"
• Abstract: "For any a,b,c: (a+b)×c"

In ML:
• Specific: "This CNN classifies cats"
• Abstract: "Function approximator f: X → Y"
```

---

## 🌍 Levels of Abstraction in ML

| Level | Example |
|-------|---------|
| Implementation | PyTorch tensor operations |
| Algorithm | Gradient descent |
| Model | Neural network |
| Theory | Universal approximation |
| Mathematics | Function spaces |

---

## 💡 Why It Matters

```
Good abstraction:
• Reveals common patterns
• Enables transfer of ideas
• Simplifies complex systems

Bad abstraction:
• Hides important details
• Creates false equivalences
```
