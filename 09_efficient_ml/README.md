## 💡 Key Takeaways

<table>
<tr>
<td>

### 📊 Roofline Model
```
Perf = min(Peak_Compute, 
           Peak_BW × Intensity)
```

</td>
<td>

### 🔢 Quantization
```
FP32 → INT8: 4x smaller
FP32 → INT4: 8x smaller
2-4x speedup
```

</td>
<td>

### ⚡ Flash Attention
```
Standard: O(N²) memory
Flash: O(N) memory
5x faster!
```

</td>
</tr>
</table>
