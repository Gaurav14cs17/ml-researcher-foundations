## 💡 Key Formulas

<table>
<tr>
<td>

### 🧠 Forward Pass
```
z = Wx + b
a = σ(z)
```

</td>
<td>

### 🔄 Backprop
```
∂L/∂W = ∂L/∂a · ∂a/∂z · ∂z/∂W
```

</td>
<td>

### ⚡ Attention
```
Attn(Q,K,V) = softmax(QKᵀ/√dₖ)V
```

</td>
