## рҹ’Ў Key Algorithms

<table>
<tr>
<td>

### рҹ“ү Gradient Descent
```python
Оё вҶҗ Оё - О·вҲҮL(Оё)
```

</td>
<td>

### рҹҸғ Momentum
```python
v вҶҗ ОІv + вҲҮL(Оё)
Оё вҶҗ Оё - О·v
```

</td>
<td>

### рҹҡҖ Adam
```python
m вҶҗ ОІвӮҒm + (1-ОІвӮҒ)вҲҮL
v вҶҗ ОІвӮӮv + (1-ОІвӮӮ)(вҲҮL)ВІ
Оё вҶҗ Оё - О·В·mМӮ/(вҲҡvМӮ+Оө)
```

</td>
