# L1 and L2 Regularization

> **Controlling model complexity**

---

## 📐 Formulations

```
L2 (Ridge / Weight Decay):
L_reg = L + λ||w||₂² = L + λΣwᵢ²

L1 (Lasso):
L_reg = L + λ||w||₁ = L + λΣ|wᵢ|
```

---

## 📊 Comparison

| Aspect | L1 | L2 |
|--------|----|----|
| Sparsity | Yes (zeros out weights) | No |
| Solution | Not unique | Unique |
| Gradient | Discontinuous at 0 | Smooth |
| Use case | Feature selection | General regularization |

---

## 🔑 Why They Work

```
L2: Equivalent to Gaussian prior on weights
    p(w) ∝ exp(-λ||w||²)
    MAP estimation = L2 regularization

L1: Equivalent to Laplace prior on weights
    p(w) ∝ exp(-λ||w||₁)
    MAP estimation = L1 regularization
```

---

## 💻 Code

```python
import torch

# L2 regularization (weight decay in optimizer)
optimizer = torch.optim.Adam(
    model.parameters(),
    lr=0.001,
    weight_decay=0.01  # L2 penalty
)

# Manual L2
l2_lambda = 0.01
l2_reg = sum(p.pow(2).sum() for p in model.parameters())
loss = loss + l2_lambda * l2_reg

# Manual L1
l1_lambda = 0.01
l1_reg = sum(p.abs().sum() for p in model.parameters())
loss = loss + l1_lambda * l1_reg
```
